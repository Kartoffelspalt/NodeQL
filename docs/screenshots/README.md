# NodeQL screenshot gallery

Screenshots in this directory are used by the public project README and release
notes.

## Current screenshots

- `darkmode-preview.png`: visual SQL workbench in Dark Mode.
- `whitemode-preview.png`: visual SQL workbench in White Mode.
- `NodeQL-Einfuehrung.png`: interactive onboarding tutorial.
- `Plugins.png`: plugin blocks and Plugin SDK integration.

## Contribution guidelines

- Capture the current public release or the current `main` branch.
- Prefer a clean sample database without personal or confidential data.
- Use PNG or WebP and a descriptive lowercase filename.
- Capture the complete relevant workflow instead of isolated controls where
  possible.
- Include both Dark Mode and White Mode when a change affects theming.
- Remove local paths, credentials, tokens, user names, and private repository
  URLs before committing an image.
- Update the Gallery section in the root `README.md` when adding a highlighted
  screenshot.
